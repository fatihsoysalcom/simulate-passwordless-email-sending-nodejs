# Simulate Passwordless Email Sending Nodejs

This example demonstrates how to simulate sending passwordless login emails in a Node.js environment without actually dispatching them through an SMTP server. It captures the email content into a mock inbox, allowing developers to inspect and assert its contents for testing purposes. This approach prevents real inboxes from being cluttered during development and testing cycles.

## Language

`javascript`

## How to Run

Save the code as `simulate_email_test.js`. Open your terminal, navigate to the file's directory, and run `node simulate_email_test.js`.
